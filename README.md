# Björnsta Consulting Group

Portfolio website for Björnsta Consulting Group - an accounting and tax services firm in Eskilstuna, Sweden.

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Netlify Forms

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
```

## Deployment

Configured for Netlify deployment. Push to main branch to trigger automatic deployment.
